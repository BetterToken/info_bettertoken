# Titan Node V2
The Titan Node is a medium sized node that already packs quite a punch and will be able to run heavier workloads.


## Specifications

* AMD Ryzen 5 3400G (8 logical cores)
* CPU Passmark: 9,400
* 32 GB Ram
* 480 GB SSD 
* 512 GB HDD

## Dimensions

* Width 176 mm
* Height 200 mm
* Depth 380 mm
