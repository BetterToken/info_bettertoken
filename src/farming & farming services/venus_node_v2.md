# Venus Node V2
The Venus node is the mother of the smaller 3nodes.
She comes in around the same size of a small pc tower and packs a lot of power!


## Specifications

* AMD Ryzen 9 3600x (24 logical cores)
* CPU Passmark: 32,800
* 128 GB Ram
* 2 GB SSD 
* 2 x 7 TB HDD

## Dimensions

* Width 203 mm
* Height 330 mm
* Depth 400 mm
