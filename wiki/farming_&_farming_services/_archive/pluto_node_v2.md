# Pluto Node V2
The Pluto Node V2 is the smallest of the bunch. Coming in the same casing as the Titan V2 you could regard it as it's smaller brother.

## Specifications

* AMD Ryzen 3 3200G (4 logical cores)
* CPU Passmark: 7,200
* 8 GB Ram
* 480 GB SSD 

## Dimensions

* Width 176 mm
* Height 200 mm
* Depth 380 mm
